## 개요
* 무슨 사이트인가요?

광주고 내부의 지도와 dijkstra 알고리즘을 이용하여 광주고등학교내의 길을 찾아드립니다.
* 왜 만들었나요?
  
광주고등학교를 처음 방문하는 사람들이 길을 못찾는 경우가 자주 있어서 이를 해결하기 위하여 만들었습니다.
